# حروف مع سيف (7rof)

لعبة "حروف مع سيف" هي لعبة تفاعلية تعتمد على شبكة عرافية (Hex Grid) حيث يتنافس فريقان (الأحمر والأزرق) للسيطرة على الحروف عبر الإجابة على الأسئلة.

## المميزات
- شبكة حروف تفاعلية.
- نظام غرف (Rooms) للعب المتعدد.
- أسئلة متنوعة (أكثر من 500 سؤال).
- واجهة تحكم للمضيف وواجهة للاعبين.

## التشغيل المحلي
1. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```
2. ابدأ السيرفر:
   ```bash
   npm start
   ```

## النشر (Deployment)
يمكن نشر هذه اللعبة بسهولة على منصات مثل **Render** أو **Railway**.
1. ارفع الكود على مستودع (Repository) في GitHub.
2. اربط المستودع بمنصة Render.
3. اختر `npm start` كأمر التشغيل (Start Command).
